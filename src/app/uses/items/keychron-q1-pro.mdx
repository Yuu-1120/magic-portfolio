## Overview

The Keychron Q1 Pro is a fully customizable 75% mechanical keyboard that I use as my primary input device. With its aluminum body, hot-swappable switches, and QMK/VIA support, it strikes the perfect balance between typing feel, programmability, and wireless convenience.

## Specifications

- **Layout:** 75% (82 keys)
- **Switches:** Gateron Oil King (linear, pre-lubed)
- **Keycaps:** Double-shot PBT, Cherry profile
- **Case:** CNC aluminum, black
- **Connectivity:** Bluetooth 5.1, USB-C wired
- **Battery:** 4000mAh (up to 300 hours)
- **Firmware:** QMK / VIA compatible

## Why This Keyboard

### Typing Experience

The Gateron Oil King switches are incredibly smooth straight out of the box — no additional lubing needed. Combined with the double-shot PBT keycaps, the typing sound is a satisfying deep "thock" without being too loud for shared spaces.

### Build Quality

The CNC aluminum case gives the keyboard a solid, premium feel. It's heavy enough to stay planted on the desk but not so heavy that it's impractical to move. The gasket-mounted PCB provides a cushioned typing feel that reduces fatigue during long coding sessions.

### Programmability

With QMK/VIA support, I've customized several layers:

1. **Base layer:** Standard QWERTY with minor tweaks
2. **Code layer:** Custom shortcuts for common symbols (`{}`, `()`, `=>`, `[]`) on easy-reach keys
3. **Design layer:** Zoom controls and tool shortcuts for Figma and Photoshop
4. **Media layer:** Volume, playback, and brightness controls

### Wireless Freedom

The Bluetooth 5.1 connection is rock-solid. I can seamlessly switch between the MacBook Pro (clamshell) and iPad with a key combo. The 4000mAh battery lasts me about 3 weeks with moderate use.

## My Key Remaps

- `Caps Lock` → `Hyper` (⌃⌥⇧⌘) for custom shortcuts
- `Right Alt` → `Layer toggle` for code symbols
- `Fn + Tab` → Switch between Bluetooth devices
- `Fn + Space` → Toggle between Mac and Windows key layout

## Maintenance

- Clean keycaps monthly with compressed air
- Re-lub stabilizers every 6 months
- Firmware updates via VIA web app when needed
