## Overview

The MacBook Pro 16" with M3 Max chip has been my primary development and design machine since early 2024. It handles everything I throw at it — from running local dev servers and Docker containers to Figma with dozens of artboards open simultaneously.

## Specifications

- **Chip:** Apple M3 Max (16-core CPU, 40-core GPU)
- **Memory:** 64GB unified memory
- **Storage:** 2TB SSD
- **Display:** 16.2-inch Liquid Retina XDR

## Why This Machine

After years of using various laptops, the M3 Max MacBook Pro checks every box:

### Performance

The unified memory architecture means no more swapping when running Next.js dev server, a local Supabase instance, Figma, and 50+ browser tabs simultaneously. Cold starts on large TypeScript projects finish in seconds.

### Display

The Liquid Retina XDR display with ProMotion is incredible for design work. Colors are accurate out of the box, and the 120Hz refresh rate makes scrolling through code feel buttery smooth.

### Battery Life

Despite the power, I regularly get 12+ hours of real-world usage. This means I can leave the charger at home for most days.

## My Setup

I typically run it in clamshell mode connected to my Apple Studio Display, with the Keychron Q1 Pro and MX Master 3S as peripherals. When mobile, the built-in keyboard and trackpad are both excellent.

## Daily Workflow

1. **Morning:** Open Cursor, pull latest changes, review PRs
2. **Design:** Switch to Figma for UI work — runs natively on Apple Silicon
3. **Development:** Run `next dev` with Turbopack, local Supabase, and testing suites
4. **Evening:** Light browsing, reading documentation, or sketching ideas in Procreate (via Sidecar with iPad)
